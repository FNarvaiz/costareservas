Instalación

El instalador descargará los componentes automaticamente y los instalará. Cuando se suba una nueva versión del sistema con mas funcionalidades, se actualizará solo antes de iniciar. El programa es gratuito por tiempo indeterminado de prueba.

Requisitos

· Sistemas operativo: Windows Vista / 7 / 8 / 10

· SQL Server 2008 R2 Express

· NET Framework 4.5.2 (x86 y x64) Aquí

· Report Viewer 2012 Runtime Aquí

Si ya tiene instalados estos componentes, puede iniciar ahora la aplicación. De lo contrario, haga clic en el botón de abajo para instalar los requisitos previos y ejecutar la aplicación.
